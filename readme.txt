== Description ==

### Be Inspired

Feeling inspired and being highly motivated can help elevate new content creation. Install this plugin and each page you see while logged into your WordPress dashboard will show you a new inspirational quote in the top right corner of the dashboard.

Whether you're blogging to make money or just want to write the most incredible articles for readers, don't leave any ounce of motivation behind. Once activated this plugin automatically works and there's no need to worry about any settings.

== Installation ==

= Using The WordPress Dashboard =

1. Navigate to the 'Add New' section in the plugins dashboard
1. Search for 'wpadmin-motivation'
1. Click on the 'Install Now' button
1. Activate the plugin in the Plugins dashboard and see a quote in the top right corner of your dashboard

= Using Manual Upload =
1. Upload the zipped plugin folder to your 'wp-content/plugins/' directory
1. Activate the plugin using the 'Plugins' page in your WordPress dashboard and see an inspirational quote in the top right corner of your dashboard

== Frequently Asked Questions ==

= Will the quotes show on the front end of my website? =
No. They are only visible in the WordPress dashboard.

= Where is the settings page located? =
There isn't one required for this plugin, it works automatically and there's no settings to change. This also helps keep the plugin light and avoid bloat.

= How can I add my own quotes? =
Please fork the plugin to add your own quotes.
